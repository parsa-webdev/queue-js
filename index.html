A Simple Component

Queue components implement a render() method that takes input data and returns what to display. This example uses an XML-like syntax called JSX. Input data that is passed into the component can be accessed by render() via this.props.

This example uses an XML-like syntax called JSX. Input data that is passed into the component can be accessed by render() via this.props.

Code

            
class HelloMessage extends Queue.Component {
    render() {
        return (
        <div>
            Hello {this.props.name}
        </div>
        );
    }
    }
    
    QueueDOM.render(
        <HelloMessage name="John" />,
        document.getElementById('hello-example'));

Result

Hello John
